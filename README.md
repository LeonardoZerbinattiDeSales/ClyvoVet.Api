# 📚 Objetivo do Projeto

O objetivo do projeto é desenvolver uma API RESTful para gerenciamento clínico veterinário, permitindo o cadastro e gerenciamento de:

- Tutores
- Pets
- Eventos Clínicos

O projeto foi desenvolvido seguindo os requisitos da disciplina **Advanced Business Development with .NET**.
